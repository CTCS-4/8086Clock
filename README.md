## 外围芯片部分简要说明

初步设想由8253的通道0以方式0，每1s提供一次中断，经由8259调用中断服务程序TICK（在业务逻辑部分实现）；8255连接到外部调节按钮和LED数码管；8259另有一个优先级更高的中断与设置按钮连接，可触发初始时间设置程序INIT。

拟分配实验板上IR1给INIT，IR2给TICK。
